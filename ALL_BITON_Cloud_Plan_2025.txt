Cloud Infrastructure Plan - ALL BITON
📋 תקציר / Executive Summary
עברית: מסמך זה מתעד תכנון מלא למערך ענן היברידי המשלב AI, כרייה, שירותי אינטרנט ותקשורת. תקציב: 1,800 ₪ לחודש (~552$). ניצול קרדיטים ב-Vast.ai (382$) ו-Google Drive (30TB).

English: This document outlines a comprehensive hybrid cloud infrastructure plan combining AI, mining, web services, and communication. Budget: 1,800 ILS/month (~552$). Utilizing existing Vast.ai credits (382$) and Google Drive (30TB).

⚖️ חוקיות / Legality
🔵 Alibaba Cloud
אסור: כריית קריפטו, שימוש אינטנסיבי במשאבים

מותר: שרתי אינטרנט, AI (Model Studio), APIs, ניהול

סיכון: סגירת חשבון בהפרת תנאים

🟢 Vast.ai
מותר: כריית קריפטו עם GPU

דרוש: מימון במטבע קריפטו (לא קרדיטי אשראי)

חוקי: בהתאם לחוקי המדינה ומדיניות המארח

🇮🇱 רגולציה בישראל
מעמד: כרייה חוקית

מס: 25% רווח הון (פרטי) / עד 50% + 17% מע"מ (עסק)

דיווח: מעקב רווחים, ייעוץ מס מומלץ

🚀 שלבי יישום / Implementation Steps
📅 שלב 1: ימים 0-3
הרשמה ל-Free Tier של Alibaba Cloud

פריסת שרתי A1-A4 עם Docker

התקנת Grafana, Prometheus, PostgreSQL

הגדרת אבטחה בסיסית (Firewall, SSH Keys)

📅 שלב 2: ימים 4-7
המימון Vast.ai עם קריפטו

הפעלת V1 (RTX 5090) לניסוי כרייה

אינטגרציית ניטור עם Grafana

התקנת Abraham-AI על שרת A1

📅 שלב 3: שבוע 2
בדיקת רווחיות כרייה (Flux/Qubitcoin)

הגדרת שרת משחקים על V2

חיבור Google Drive לגיבויים

בניית API לשירותי AI

📅 שלב 4: חודש 1+
אופטימיזציה לפי ביצועים

הרחבת GPU לפי רווחיות

פיתוח שירותים נוספים

מוניטיזציה של פלטפורמה

⚠️ אזהרות קריטיות / Critical Warnings
